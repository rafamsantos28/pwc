# Razonne+ — Plataforma de Streaming

Este repositório contém o projeto final desenvolvido para a unidade curricular de **Programação Web - Cliente** do 1.º Ano do CTeSP de Desenvolvimento para a Web e Dispositivos Móveis do ISLA Gaia.

O objetivo principal deste projeto é aplicar de forma prática os conceitos de estruturação semântica em HTML5, estilização e responsividade com Bootstrap 5, otimização básica de SEO e organização limpa de código.

---

## 🎯 Objetivo do Projeto

O website foi concebido sob a vertente de Marketing para uma Empresa, promovendo o serviço fictício de streaming Razonne+. A página funciona como uma *Landing Page* de conversão, cujo propósito é apresentar os diferenciais tecnológicos da plataforma e exibir destaques do catálogo.

---

## 🚀 Tecnologias Utilizadas

Para garantir a total conformidade com as diretrizes e restrições estabelecidas para o trabalho, foram utilizadas as seguintes tecnologias:

*   **HTML5 Semântico:** Utilização rigorosa de elementos estruturais para acessibilidade e organização (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`).
*   **Bootstrap 5:** Implementação do sistema de *Grid* (linhas e colunas), utilitários de espaçamento, tipografia fluida e componentes responsivos nativos.
*   **CSS3 Customizado:** Sobrecrita cirúrgica de componentes do Bootstrap para alinhar a palete de cores do layout à identidade visual *dark/neon* da marca.
*   **Font Awesome:** Iconografia vetorial para suporte visual na área de funcionalidades e formulários.
*   **Google Fonts:** Incorporação das famílias tipográficas Montserrat (para títulos de forte impacto comercial) e Inter (para leitura fluida de textos de apoio).

---

## 📦 Requisitos Mínimos Cumpridos

O projeto preenche na totalidade os parâmetros de avaliação exigidos no enunciado:

1.  **Header com Navegação Funcional:** Menu superior fixo (`sticky-top`) com efeito de desfoque de fundo e colapso em botão hambúrguer totalmente nativo para dispositivos móveis.
2.  **Secção Hero Bem Definida:** Zona de conversão primária com títulos hierárquicos fortes, proposta de valor clara e botões de chamada para ação.
3.  **Área de Features (Vantagens):** Disposição em grelha de 3 colunas utilizando `<article>` para destacar os pilares técnicos do serviço.
4.  **Secção de Contacto:** Formulário de suporte integrado e estilizado, utilizando controlos de validação nativos de formulários do Bootstrap.
5.  **Footer Consistente:** Encerramento estruturado da página com direitos de autor e notas de contextualização.
6.  **Responsividade Total:** Fluidez garantida através de *breakpoints* do Bootstrap, oferecendo uma experiência de navegação otimizada em computadores, tablets e telemóveis.
7.  **SEO Básico Aplicado:** Inclusão de tags cruciais como `meta description` otimizada, palavras-chave estruturadas, definição correta do idioma e hierarquia semântica de títulos (`<h1>` a `<h3>`).
